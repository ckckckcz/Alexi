Apa Itu Bahasa Pemrogaman CSS ?, Pengertian, Sejarah, Framework, dan Bagaimana Cara Penggunaanya

Artikel ALEXI – CSS Merupakan Salah Satu Bahasa Pemograman Yang Khusus Digunakan Untuk Mendesain Atau Memperindah Tampilan Website , Pada Artikel Ini ALEXI Akan Membahas Mengenai Apa Itu Bahasa Pemrogaman CSS ?, Pengertian, Sejarah, Jenis Framework, dan Bagaimana Cara Penggunaanya

* Pengertian

Cascading Style Sheet Atau Bisa Dikenal Dengan CSS, Merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. Pada umumnya CSS digunakann untuk memformat tampilan halaman web yang dibuat dengan bahasa HTML dan XHTML.

CSS dapat mengendalikan ukuran gambar, warna bagian tubuh pada teks, warna tabel, ukuran border, warna border, warna hyperlink, warna mouse over, spasi antar paragraf, spasi antar teks, margin kiri,kanan, kanan, bawah, dan parameter lainnya.

* Sejarah

Nama CSS didapat dari fakta bahwa setiap deklarasi style yang berbeda dapat diletakkan secara berurutan, yang kemudian membentuk hubungan seperti orang tua-anak (parent-child) pada setiap style.

Untuk saat ini terdapat tiga versi CSS, yaitu CSS 1, CSS 2, dan CSS 3. CSS 1 dikembangkan berpusat pada pemformatan dokumen HTML, CSS 2 dikembangkan untuk memenuhi kebutuhan terhadap format dokumen agar bisa ditampilkan di printer, sedangkan CSS 3 adalah versi terbaru dari CSS yang mampu melakukan banyak hal dalam desain website. CSS 3 juga mendukung penentuan posisi konten, downloadable, huruf font, tampilan pada tabel /table layout dan media tipe untuk printer. Kehadiran versi CSS yang ketiga diharapkan lebih baik dari versi pertama dan kedua.

CSS 3 juga dapat melakukan animasi pada halaman website, di antaranya animasi warna hingga animasi 3D. Dengan CSS 3 desainer lebih dimudahkan dalam hal kompatibilitas websitenya pada smartphone dengan dukungan fitur baru yakni media query. Selain itu, banyak fitur baru pada CSS3 seperti: multiple background, border-radius, drop-shadow, border-image, CSS Math, dan CSS Object Model.

* Cara Penggunaan

Dalam Penggunaan CSS Ada 3 Cara yaitu, Inline Css, Internal Css, Dan External Css. Berikut Untuk Penjelasannya.

1. Inline Css

Inline CSS digunakan untuk memberi style (gaya) pada tag atau elemen HTML tertentu. Kamu dapat menggunakan atribut style untuk memberi style pada tag HTML. Style hanya berlaku untuk tag HTML tersebut. Style di inline CSS memiliki prioritas utama dan yang akan digunakan (rendering) meskipun tag HTML tersebut juga sudah diberi style melalui internal CSS atau external CSS.

Pengunaan inline CSS sebenarnya tidak direkomendasikan karena setiap tag HTML perlu diberi style sendiri. Ketika Anda mengubah style di salah satu tag untuk mendapatkan style yang konsisten, Anda juga perlu mengubah style pada tag-tag lainnya. Misalnya, semua tag h4 berwarna hijau, ketika Anda ingin mengubah tag h4 menjadi berwarna biru, Anda perlu mengubah style di semua h4 satu persatu agar berwana biru. 2. Internal Css

Style yang diberikan diapit oleh tag <style></style> dan diletakkan di bagian elemen head atau tag <head> dari laman web. Kamu dapat menggunakan internal CSS bila pemberian style dimaksudkan hanya untuk satu laman web saja dan tidak untuk digunakan di laman web yang lain. Kamu dapat menggunakan internal CSS bila kode untuk style tidak terlalu besar atau kompleks. Bentuk contoh kode sederhana sebagai berikut:

3. External Css

External CSS merupakan file berekstensi .css yang hanya berisi kode-kode style. Tujuan menggunakan external CSS adalah untuk memisahkan kode style secara terstruktur. External CSS dapat diaplikasikan pada lebih dari satu laman web sehingga semua laman web akan memiliki tampilan atau desain yang seragam.

Sebagai contoh, buat file HTML dengan nama index.html, dan buat juga file CSS dengan nama style.css menggunakan plain text editor kesayangan kalian seperti Notepad++, Visual Studio Code, Dan Sublime Text.

Kemudian di file HTML dan di bagian tag <head> tambahkan elemen link atau tag <link>. Pada Tag <link> inilah yang akan digunakan untuk menghubungkan file HTML ke file CSS yang dibuat sebelumnya. Perhatikan penulisan <link> beserta atributnya untuk mengacu ke file CSS. Berikut contoh penulisan codenya, Selanjutnya, di file CSS tuliskan code CSS di bawah ini sebagai style yang akan diberlakukan pada tampilan web. Simpan file CSS di folder yang sama dimana kamu menyimpan file HTML tersebut.
